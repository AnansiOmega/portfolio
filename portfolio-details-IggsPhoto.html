Igg's Photo Shop

Igg's Photo Shop is a mock e-commerce Photo Shop where a User can add photo's to their cart and 'purchase' them. Instead of actually buying anything the user is re-directed to a website where they can make donations to me. The application has support for a paypal button that is fully functional. But, because this is one of many projects and I didn't find it worthwhile to set up a business account for an application that strictly takes donations. Furthermore, this application was built entirely with React & Redux with functional components to give it speed without having to sacrifice any of the lifecycle methods a class component may have.
