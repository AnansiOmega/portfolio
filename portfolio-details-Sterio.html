Ster.io

Ster.io is a web application built for musicians. A User must have an account to use this application, and is unable to make an account with falsey information. Once the User is signed up/in they will be redirected to the home page that lists all of the websites tracks. These tracks are meant to be built upon. Once a single track has a collaboration then it is considered a song. But it doesn't stop there, songs can also be collaborated with to grow larger. Every time a song or track gets collaborated the original track/song gets preserved so other users can add on to differen versions. My vision for this application was to build a playground for musicians to jam together over the web and I'm incredibly satisfied with how it turned out.
