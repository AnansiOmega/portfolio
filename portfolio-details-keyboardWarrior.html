Keyboard Warrior

Keyboard Warrior is an application where a user inputs a title and a track by an artist. Once selected, the lyrics will be displayed onto the screen the user will begin typing each line. Once it is typed without mistakes the next line is showed, etc.. There is a real time WPM display in the lightbulb as well as the bottom left corner of the screen. Once the user has completed their song, they will be told how quickly they were able to type out the song. This whole application was built using strictly React with hooks.
